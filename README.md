> **Курсовая работа**  
> *Тема: Разработка веб-приложения для планирования путешествий*

Современное full-stack веб-приложение для поиска, бронирования и организации туров. Платформа объединяет путешественников и организаторов: пользователи могут находить туры по странам, категориям и сезонам, а организаторы — публиковать собственные маршруты.

---

## Возможности

### Для путешественников
- Главная страница с категориями, популярными и рекомендуемыми турами
- Каталог туров с фильтрами по стране, категории, сезону, сложности и цене
- Поиск туров по названию и описанию
- Детальная страница тура с программой по дням, включениями/исключениями и отзывами
- Добавление туров в избранное
- Корзина и оформление бронирования
- Личный кабинет с историей бронирований

### Для организаторов
- Создание и публикация туров
- Управление опубликованными маршрутами

### Администрирование
- Просмотр статистики по турам, категориям и пользователям
- Модерация публикаций
- Управление пользователями

## Технологический стек

| Категория | Технологии |
|-----------|------------|
| **Frontend + Backend** | Next.js 14 (App Router), TypeScript |
| **Стили** | Tailwind CSS |
| **База данных** | SQLite (Prisma ORM) |
| **Аутентификация** | JWT-токены (jose) |
| **Управление состоянием** | Zustand |
| **Иконки** | lucide-react |

## Структура проекта

```
.
├── app/                    # Страницы и API routes (Next.js App Router)
│   ├── api/               # REST API endpoints
│   │   ├── auth/          # Авторизация (login, register, logout, me)
│   │   ├── trips/         # Туры и маршруты
│   │   ├── categories/    # Категории туров
│   │   ├── bookings/      # Бронирования
│   │   └── organizer/     # Endpoints для организаторов
│   ├── admin/             # Админ-панель
│   ├── catalog/           # Каталог туров
│   ├── trip/[slug]/       # Страница тура
│   ├── create-trip/       # Создание тура
│   ├── cart/              # Корзина
│   ├── checkout/          # Оформление бронирования
│   ├── profile/           # Личный кабинет
│   └── page.tsx           # Главная страница
├── components/            # React-компоненты
├── lib/                   # Утилиты (Prisma, auth, utils)
├── prisma/                # Схема БД и seed-данные
│   ├── schema.prisma
│   └── seed.ts
├── store/                 # Zustand store (корзина, избранное)
├── types/                 # TypeScript типы
├── public/                # Статические файлы и изображения
│   └── images/
│       ├── trips/         # Изображения туров
│       └── placeholders/  # Заглушки
└── README.md
```
