# TravelPlan

Курсовая работа по теме "Разработка веб-приложения для планирования путешествий".

## Описание

Веб-приложение для поиска, бронирования и организации туров. Пользователи могут просматривать каталог туров, фильтровать их, добавлять в избранное и корзину, оформлять бронирование. Организаторы могут публиковать свои туры. Администратор управляет пользователями, турами и заказами.

## Технологии

- Next.js 14
- TypeScript
- Tailwind CSS
- Prisma ORM
- SQLite
- JWT (jose)
- Zustand

## Запуск

```bash
npm install
npm run db:generate
npm run db:reset
npm run dev
```

После запуска открыть http://localhost:3000.
